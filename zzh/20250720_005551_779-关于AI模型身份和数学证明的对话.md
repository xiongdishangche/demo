**皮亚诺公理体系证明**

在自然数公理体系中，1+1=2可以通过以下步骤证明：

1. **定义**：1是自然数集合中的第一个非零元素
2. **后继函数**：S(n)表示n的后继数
3. **加法定义**：
   - 对于任意自然数n，n + 0 = n
   - 对于任意自然数m和n，m + S(n) = S(m + n)
4. **证明过程**：
   - 1 + 1 = 1 + S(0) （因为1是0的后继）
   - = S(1 + 0) （根据加法定义）
   - = S(1) （因为1 + 0 = 1）
   - = 2 （因为2是1的后继）

**集合论证明**

- 设A = {a}，B = {b}，其中a ≠ b
- A ∪ B = {a, b}
- |A| = 1，|B| = 1
- |A ∪ B| = 2
- 因此1 + 1 = 2

在日常生活中，如果我们有一个苹果，再拿一个苹果，我们就有两个苹果。这种直观经验支持了1+1=2的正确性。
